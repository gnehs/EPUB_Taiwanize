# EPUB_Taiwanize
  
Convert EPUB from zh_CN to zh_TW using [ZHConvert](https://zhconvert.org/)

透過[繁化姬](https://zhconvert.org/)將 EPUB 電子書台灣化

## Usage 使用
`python3 main.py YOUR_EPUB_FILE`
